# Texas State Technical College Brand Guidelines

## Brand Colors

### Primary Colors
- **Texas Blue**: `#1d2757`
  - Primary brand color for headers, navigation, and main UI elements
  - Use for trust, stability, and educational authority
  
- **Lone Star Red**: `#d13138`
  - Secondary brand color for accents, calls-to-action, and highlights
  - Use for emphasis, alerts, and interactive elements

### Color Variations
- **Texas Blue Light**: `#2a3a7a` (for hover states, lighter applications)
- **Texas Blue Dark**: `#141f42` (for pressed states, emphasis)
- **Lone Star Red Light**: `#e14850` (for hover states, lighter applications)
- **Lone Star Red Dark**: `#b02229` (for pressed states, emphasis)

## Usage Guidelines

### Do's
- Use Texas Blue for primary navigation and headings
- Use Lone Star Red for call-to-action buttons and important highlights
- Maintain sufficient contrast ratios for accessibility
- Use brand colors consistently across all educational materials

### Don'ts
- Don't use purple, blue, or red colors outside the brand palette
- Don't use brand colors at low opacity that compromise readability
- Don't mix brand colors with competing color schemes

## Implementation

### Quick Start
1. Link to `brand-config.css` in your project
2. Use CSS custom properties: `var(--tstc-texas-blue)` and `var(--tstc-lone-star-red)`
3. Apply predefined classes: `.tstc-primary`, `.tstc-secondary`, etc.

### For AI Projects
Include this instruction in your prompts:
*"Use Texas State Technical College brand colors: Texas Blue (#1d2757) as primary and Lone Star Red (#d13138) as secondary. Avoid purple or other brand colors."*

## Educational Content Applications

### Physics/Science Materials
- Use Texas Blue for vector arrows and main diagram elements
- Use Lone Star Red for highlights, important measurements, or key concepts
- Maintain high contrast for educational clarity

### Interactive Elements
- Primary buttons: Texas Blue background
- Secondary/action buttons: Lone Star Red background
- Active states: Use darker variations
- Borders and outlines: Brand colors at full opacity
